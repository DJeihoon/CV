# Assets

این پوشه محل قرارگیری تصویر پروفایل است.

## تصویر پروفایل

**⚠️ ضروری قبل از آپلود:** فایل `profile.jpg` را در این پوشه قرار دهید.

- **توصیه:** ابعاد حداقل 400×400 پیکسل
- **فرمت:** JPG یا PNG
- **حجم:** برای بارگذاری سریع‌تر، کمتر از 200KB توصیه می‌شود
- در صورت نبود فایل، حروف «DJ» به‌عنوان fallback نمایش داده می‌شود
